RPN Calculator
Программа для вычисления арифметических выражений в Обратной Польской Нотации (Reverse Polish Notation).


Описание
Программа читает текстовый файл input.txt в каждой строке которого записано выражение в RPN формате, вычисляет результаты и записывает их в файл output.txt.


Пример использования
input.txt:
3 2 - 1 + 2 12 6 / -
output.txt:
3 2 - 1 + = 2 2 12 6 / - = 0


Сборка и запуск
Требования
1. Компилятор С++
2. Утилита make


Инструкция
1. Клонируйте репозиторий:
git clone https://github.com/your-username/your-repo-name.git cd your-repo-name
2.  Соберите проект
make
3.  Создайте файл input.txt с выражениями и запустите программу:
make run


Доступные команды Makefile
make              - скомпилировать программу"
make run          - запустить программу"
make test_input   - создать тестовый input.txt"
make show_input   - показать input.txt"
make show_output  - показать output.txt"
make quick_test   - сделать всё: очистить, создать тест, скомпилировать, запустить"
make clean        - удалить исполняемый файл и output.txt"
make clean_all    - удалить все созданные файлы"
make help         - показать справку


Формат выражений
Числа: целые или дробные (например: 5, 3.14)
Операторы: +, -, *, /
